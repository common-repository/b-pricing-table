Create and display product professional-looking pricing table in WordPress.

== Description ==

Create and display product professional-looking pricing tables in WordPress's block editor. This is a block plugin and it will only work with the Gutenberg editor. 


**[Demos](https://bblockswp.com/demo/pricing-table/)**


= Features =
- **Fully Customizable**: All the options you need to arrange the showcase to your liking are available here.


= How to use =
- First, install the B Pricing Table plugin
- Add the B Pricing Table from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab.

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **B Pricing Table**
3. Click on the **Pricing Table** to add the block

= Download & Upload: =
1. Download the **B Pricing Table** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`b-pricing-table.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **B Pricing Table** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
